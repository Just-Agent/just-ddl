# Just-DDL

Just-DDL is the hub for deadline-tracking topics. Each major topic can live in its own GitHub repository, with its own GitHub Pages site, Actions workflow, data source, and future mini program or app surface.

- Live site: https://just-agent.github.io/just-ddl/
- Repository: https://github.com/Just-Agent/just-ddl
